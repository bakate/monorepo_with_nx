## Objects

<Image src="phones.jpg" />

Objects are the basic building blocks of the game. They are the things that you can interact with.


```ts
// Create a new object
interface ObjectProps {
  name: string;
  description: string;
  image?: string;
  tags?: string[];
}
const obj:ObjectProps = {
  name: 'My Object',
description: 'This is my object',
}

console.log(obj.name) // Logs: My Object
```
